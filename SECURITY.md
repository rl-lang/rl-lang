# Security Policy

## Supported Versions

| Version | Supported |
| ------- | --------- |
| 0.1.x   | ✅        |

## Reporting a Vulnerability

If you find a security vulnerability in rl-lang, please report it by opening a [GitHub Issue](https://github.com/MohamedGonem/rl-lang/issues) or contacting me directly via GitHub.

I'll respond as soon as possible. If the vulnerability is valid I'll work on a fix and credit you in the release notes.
